        本地管理

        设置

        外观

        暗黑护眼

        默认跟随 Windows 深色模式；暗黑护眼使用低亮度灰黑界面，不反色 PDF / OFD / 图片内容。

        最近文件

        只保存本机最近打开记录，关闭后不会继续记录新的最近文件。

        语言

        界面语言

        当前阶段保存语言偏好，默认中文；完整英文界面将在后续版本逐步补齐。

        缓存

        清理本地渲染缓存，不会删除原始文档。

        OFD

        OFD 性能设置

        Office / WPS

        使用用户显式配置的本地 LibreOffice 路线；不会自动探测或随包安装，不做原格式编辑。

        用于 Office / WPS 预览。Windows 通常填写 soffice.exe，Linux 通常填写 /usr/bin/libreoffice 或 /usr/bin/soffice；本地转换只在本机进行，不会上传文件。

        隐私边界

        本工具不上传文件、不默认联网、不采集文档内容；最近文件、缓存和诊断信息都只围绕本地预览工作。

        离线说明

        帮助

        支持格式

        当前版本聚焦本地预览：OFD、PDF、Office/WPS、txt / log / csv / md 文本和常见图片。

        Office / WPS 边界

        Office / WPS 预览依赖本地 LibreOffice 转换路线。LibreOffice 是开源办公套件；用户安装并配置程序路径后，转换在本机完成，不会联网或上传文件。本工具不做原格式编辑，也不随应用安装转换组件。

        打印边界

        当前默认使用 WebView2 原生系统打印入口；这是基础打印入口，不承诺完整页码范围或跨设备矩阵。

        OFD 边界

        OFD 提供稳定模式、扩展模式和长文档实验模式。超过当前模式保护阈值时，会提示用户调整设置后再打开。

        外观边界

        暗黑护眼只调整阅读器外壳亮度；不反色 PDF / OFD / 图片内容，也不改变打印输出边界。

        隐私与诊断

        最近文件、缓存、诊断只服务本机阅读体验；诊断用于复制错误上下文，用户自行决定是否分享。

        问题反馈

        遇到问题时，可通过 GitHub Issues 手动反馈。提交前请先移除本机路径、账号信息和文档内容，不要上传真实或敏感文档。

        检查更新

        检查更新属于用户主动触发的联网动作；当前版本暂未接入自动更新源。请关注 GitHub Releases 手动获取新版：https://github.com/LocalDoc-Viewer/local-doc-viewer/releases

        关于

        local-doc-viewer 是本地优先的文档预览中心，当前聚焦 OFD、PDF、Office/WPS 本地转换预览、文本和图片阅读。
